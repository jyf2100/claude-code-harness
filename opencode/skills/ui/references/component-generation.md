# Component Skill

生成基于 shadcn/ui 的 UI 组件的技能。
首先确认 `../ui-skills.md` 和 `../frontend-design.md` 的适用条件。

---

## 触发短语

- 「创建 hero 区域」
- 「创建价格表」
- 「创建页脚」
- 「创建联系表单」
- 「生成 UI 组件」

---

## 功能

- shadcn/ui 组件自动安装
- 响应式支持
- 可访问性支持
- Tailwind CSS 定制

---

## 支持的组件

- Hero 区域
- Pricing（价格表）
- Features（功能介绍）
- Contact（联系）
- Footer
- Header/Navigation
