# NotebookLM Slide Design YAML (2 Variations)

## 目的

在 NotebookLM 的「幻灯片资料」生成中，**用 YAML 明确设计方向以提高质量和可复现性**。

此技能为了即使相同来源也能改变外观，输出**意图理解不同的 2 个方案**YAML。

参考（使用方法/示例）：
- `https://note.com/yoshifujidesign/n/nd9c8db0b55b8`
- `https://note.com/yoshifujidesign/n/n7412bccb5762`

---

## 使用方法（NotebookLM 侧）

在 NotebookLM 加载资料来源后，
在［幻灯片资料］→［自定义幻灯片资料］→［请说明要创建的幻灯片］
中**粘贴此技能输出的 YAML 全文**然后［生成］。

---

## 输入（向用户确认）

尽可能先确认以下最多 5 项，无回答时用括号中的默认值继续。

1. 目的（例：内部共享/销售/招聘/面向投资者）（内部共享）
2. 目标读者（例：管理层/开发/非工程师）（非工程师混合）
3. 基调（例：信任感/创新/亲切）（信任感）
4. 品牌元素（logo/品牌色/字体指定）（无指定）
5. 照片/图表比例（照片多/数据多）（数据多）

---

## 输出（必须格式）

必须输出以下 **2 个方案**。

- 方案 A：**可读性、信任感、数据传达最优先**（极简/企业）
- 方案 B：**情绪、故事、留白美学优先**（编辑/生活方式）

都用**中文 YAML**，可直接粘贴到 NotebookLM。

输出格式：

- `### 方案 A` 下放 1 个 YAML（代码块）
- `### 方案 B` 下放 1 个 YAML（代码块）

---

## 生成规则

- 具体颜色包含 **HEX**（例：`#FFFFFF`）
- 也写一些「不要做的事」（例：禁止过多边框、禁止过度阴影、限制颜色数）
- 章节结构/导航/留白/网格等，**明确写出影响可复现性的约束**
- 即使内容（来源）不明也能成立，写成**结构模板**

---

## 输出模板（生成示例的骨架）

### 方案 A（极简/企业）

（根据需要，配合目的/读者/品牌色微调）

```yaml
# presentation_design_spec_minimal_corporate_cn.yaml
# 风格：现代极简 / 企业
# 目的：可读性和数据传达最优先（信任感）

全体设计设置:
  基调: "信任感、专业、简洁、逻辑性"
  配色方案:
    基础: "#FFFFFF (白) 或 #F5F5F7 (极浅灰)"
    文字色: "#111111 (近黑) 或 #333333 (炭灰)"
    强调色: "如有品牌色则使用。否则采用 #0052CC (蓝)"
    突出: "#FFB020 (琥珀) - 仅用于重要数字或注意点"
    颜色限制: "最多 3 色 + 灰度"
  排版:
    标题: "较粗的黑体（无衬线）。简短有力的词语。"
    正文: "可读性高的黑体。行距较宽。"
    数字: "无衬线粗体大字。单位（%等）稍小。"
  共同布局规则:
    网格: "12 列对齐。左右留白要宽。"
    导航: "左上小字显示 '01. 章节' 这样的编号+章节名。"
    留白: "禁止堆砌。一页幻灯片一个信息。"
    图表: "网格线/边线最小化。只对重点用强调色。"
  禁止事项:
    - "禁止过度投影阴影"
    - "禁止过多装饰边框"
    - "禁止过多渐变"

布局变体（目录）:
  - 类型: "封面"
    设计: "大留白。标题左对齐大字，副标题小字。1 条细横线。"
  - 类型: "章节首页"
    设计: "章节标题大字。背景单色极简。"
  - 类型: "结论（TL;DR）"
    设计: "结论 3 点 + 1 个大数字/指标。"
  - 类型: "双栏（课题 vs 解决）"
    设计: "竖线分割。左：课题，右：解决。标题粗体，正文细字。"
  - 类型: "数据/图表"
    设计: "柱状/折线的简洁表现。注释要短。"
  - 类型: "流程"
    设计: "水平步骤（01-05）。圆形图标最小化。"
  - 类型: "时间线"
    设计: "竖线 + 年份大 + 说明小。"
  - 类型: "总结"
    设计: "下一步行动 3 点。清单风格简洁。"
```

### 方案 B（编辑/故事）

（根据需要，配合目的/读者/照片比例微调）

```yaml
# presentation_design_spec_editorial_story_cn.yaml
# 风格：现代编辑 / 生活方式杂志风
# 目的：用留白和照片讲故事（情绪/说服）

全体设计设置:
  基调: "温和、知性、情绪、有机、精致"
  视觉识别:
    背景色: "#F3F0EB (沙米色) 或 #EBEBEB (暖灰)"
    文字色: "#333333 (炭灰) - 避免纯黑"
    强调色: "#E07A5F (陶土红) 或 #708D81 (橄榄绿)"
    图片风格:
      特征: "自然光、胶片颗粒、低饱和度"
      形状: "圆角长方形 或 拱形"
  排版:
    标题: "优雅的宋体/衬线体。重要词适度强调。"
    正文: "易读的黑体/无衬线，字稍小。"
    数字: "衬线数字，沉稳感。"
  共同布局规则:
    留白: "留白最优先。降低文字密度。"
    照片: "基本一页一张大照片。"
    边线: "禁止边框。分隔仅用留白或极细线。"
  设计规则:
    - "禁止冷色/霓虹（统一暖色调）"
    - "禁止过多图标（用照片和排版讲故事）"

布局变体（目录）:
  - 类型: "封面（杂志封面）"
    设计: "右侧照片全出血，左侧标题。细横线分隔。"
  - 类型: "引用（Quote）"
    设计: "短句居中。背景大引号淡化。"
  - 类型: "故事（1→2→3）"
    设计: "垂直步骤。点线连接。各步骤旁小圆照片。"
  - 类型: "比较"
    设计: "左右两张照片 + 各自说明。颜色克制。"
  - 类型: "数据"
    设计: "极细折线 + 说明段落补充（附加故事）。"
  - 类型: "总结"
    设计: "下一步行动 3 点。留白收尾。"
```
