# Analytics Skill

集成 Google Analytics 或 Vercel Analytics 的技能。

---

## 触发短语

- 「添加访问分析」
- 「设置 Google Analytics」
- 「添加 Vercel Analytics」
- 「想要追踪页面浏览」

---

## 功能

- 页面浏览追踪
- 事件追踪
- 转化追踪
- 用户行为分析

---

## 执行流程

1. 确认项目配置（Next.js / React 等）
2. 选择 Google Analytics 或 Vercel Analytics
3. 安装所需包
4. 生成配置文件
5. 创建事件追踪函数
