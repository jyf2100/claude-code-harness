# Component Skill

基于 shadcn/ui 生成 UI 组件的技能。
首先确认 `../ui-skills.md` 和 `../frontend-design.md` 的适用条件。

---

## 触发短语

- "创建 hero section"
- "创建价格表"
- "创建页脚"
- "创建联系表单"
- "生成 UI 组件"

---

## 功能

- shadcn/ui 组件自动安装
- 响应式适配
- 无障碍适配
- Tailwind CSS 定制

## 支持组件

- Hero section
- Pricing（价格表）
- Features（功能介绍）
- Contact（联系我们）
- Footer
- Header/Navigation
