# Deploy Setup Skill

自动化部署到 Vercel/Netlify 的技能。

---

## 触发短语

- "让我能部署"
- "想发布到 Vercel"
- "想部署到 Netlify"
- "设置生产环境"

---

## 功能

- Vercel/Netlify 项目创建指南
- 环境变量设置指南
- GitHub Actions 自动部署
- 预览部署（Pull Request）

---

## 执行流程

1. 确认项目配置
2. 选择 Vercel 或 Netlify
3. 生成设置文件（vercel.json / netlify.toml）
4. 指导环境变量设置
5. 验证部署
