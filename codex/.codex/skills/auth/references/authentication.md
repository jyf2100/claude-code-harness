# Auth Skill

使用 Clerk 或 Supabase Auth 实现认证功能的技能。

---

## 触发短语

- "添加登录功能"
- "添加认证"
- "用 Clerk 实现认证"
- "设置 Supabase Auth"
- "添加 Google 登录"

---

## 功能

- 注册/登录
- 社交登录（Google、GitHub）
- 邮箱认证
- 密码重置
- 用户资料管理

---

## 执行流程

1. 确认项目配置
2. 选择 Clerk 或 Supabase Auth
3. 安装必要的包
4. 生成认证配置文件
5. 创建登录/注册 UI
6. 设置中间件/保护路由
